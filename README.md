# MyApp

Projekt stworzony z użyciem Angular CLI (wersja 20).

## Technologie

- Angular 20 (najnowsza stabilna wersja)
- NgRx 19.2.1 (zainstalowane z flagą `--force` ze względu na brak oficjalnej kompatybilności z Angular 20)
- Angular Material
- Reactive Forms
- Angular Router
